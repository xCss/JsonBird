## 必要条件
- [Node.js](https://nodejs.org) 0.10+
- [npm](https://www.npmjs.com/) (normally comes with Node.js)
 
## 安装、运行、访问，一气呵成
``` bash
# clone
$ git clone https://github.com/xcss/JsonBird.git
# install
$ cd JsonBird && npm i
# run service
$ npm start
# open your browser,input link `http://127.0.0.1:1000`
```
